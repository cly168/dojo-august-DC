Status

The FreeBSD/ppc port is still aTier 2platform. That means it is not being fully supported by our security officer, release engineers and toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC G5 has been committed to SVN.
  • 03 March, 2008: Support for the Freescale PowerQUICC III MPC85XX family system-on-chip development boards has been committd into CVS.
  • 27 February, 2008: FreeBSD 7.0 is the first release to offically support the FreeBSD/ppc port.
